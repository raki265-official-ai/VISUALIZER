# 📊 Data Analysis & Visualization Program

A powerful **Menu-Driven Python Application** for Data Analysis and Data Visualization using **Pandas**, **NumPy**, and **Matplotlib**.

---

## 🚀 Overview

The **Data Analysis & Visualization Program** is a command-line application that helps users analyze CSV datasets with ease. It provides multiple options to explore, clean, analyze, and visualize data using an interactive menu system.

This project is ideal for students, beginners, and anyone learning Python Data Analysis.

---

# ✨ Features

## 📂 Dataset Management

- Load CSV Dataset
- View Dataset
- Display Top Records
- Display Bottom Records
- View Dataset Shape
- View Column Names
- View Data Types
- View Dataset Information

---

## 🔍 DataFrame Operations

- Select Columns
- Filter Rows
- Sort Values
- Group Data
- Find Unique Values

---

## 🧹 Handle Missing Data

- Detect Missing Values
- Fill Missing Values
- Remove Missing Values

---

## 📈 Descriptive Statistics

Generate statistics including:

- Mean
- Median
- Mode
- Maximum
- Minimum
- Standard Deviation
- Variance
- Count
- Summary Statistics

---

## 📊 Data Visualization

The program supports multiple graph types:

- 📊 Bar Plot
- 📈 Line Plot
- 🔵 Scatter Plot
- 🥧 Pie Chart
- 📉 Histogram
- 📚 Stack Plot

## 💾 Save Visualization

- Save generated graphs as image files.

# ▶️ How to Run

### Step 1

Clone the repository

```bash
git clone https://github.com/yourusername/Data-Analysis-Visualization.git
```

### Step 2

Open the project folder

```bash
cd Data-Analysis-Visualization
```

### Step 3

Install required libraries

```bash
pip install pandas numpy matplotlib
```

### Step 4

Run the project

```bash
python main.py
```
